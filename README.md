# 🎄 优美的圣诞树项目

一个精美的圣诞树动画网页，祝你圣诞快乐！

## ✨ 特性

- 🌟 闪烁旋转的金色星星
- 🎄 五层渐变绿色圣诞树，带摇摆动画
- 🔴 彩色装饰球（红、蓝、金、紫）
- ❄️ 飘落的雪花效果
- 🎁 可交互的礼物盒（点击有惊喜）
- 📱 响应式设计
- 🌈 星空渐变背景

## 🚀 使用方法

直接在浏览器中打开 `index.html` 即可查看效果。

或者启动本地服务器：

```bash
python3 -m http.server 8000
```

然后访问 http://localhost:8000/index.html

## 📁 文件结构

```
.
├── index.html    # 主页面
├── style.css     # 样式和动画
└── script.js     # 交互功能
```

## 🎮 互动功能

点击底部的礼物盒 🎁 会随机显示祝福语！
